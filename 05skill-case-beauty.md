# 多层递进Markdown思维导图渲染规范（核心分层MD）
## 语法规则（严格递进，逐级嵌套，层级不能错乱）
1. # 一级标题：业务根模块（整个需求对应一个#）
2. - 一级列表：测试大类（对应XMind一级节点）
  - 二级列表：业务子模块（缩进2空格）
    - 三级列表：场景分组（再缩进2空格）
      - 四级列表：单条用例叶子节点（再缩进2空格）
## 标准递进模板示例
# 用户手机号登录模块
- 基础正向功能测试
  - 正常登录场景分组
    - T001 合法11位手机号+有效验证码登录
      - 前置条件：账号已完成注册
      - 操作步骤：
        1. 在登录框输入11位有效手机号
        2. 点击获取验证码按钮
        3. 输入收到的6位数字验证码
        4. 点击登录按钮
      - 预期结果：登录成功，页面自动跳转首页
      - 优先级：P0
## 特性说明
1. 纯标准Markdown，支持Obsidian、ProcessOn、MindMaster、语雀全部导图工具解析
2. 天然分层递进，层级视觉区分清晰，可直接复制保存为独立md文件
3. 可单独导出，不依赖XMind，纯文档化思维导图
## 输出要求
该MD导图必须作为独立完整区块输出，层级严格逐级嵌套，禁止平级混排
